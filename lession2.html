Урок 2: Первая программа на JavaScript.

Давайте напишем нашу первую программу на JavaScript, которая будет складывать два числа и выводить результат в консоль.

1. Откройте консоль браузера:

Нажмите правой кнопкой мыши на странице, выберите "Элемент" (Inspect), затем перейдите во вкладку "Консоль" (Console).

2. Напишите программу в консоли:

Введите следующий код в консоли браузера:

        
    пример кода на javascript
    // Пример сложения двух чисел
    let первоеЧисло = 5;
    let второеЧисло = 10;
    let результат = первоеЧисло + второеЧисло;

    // Вывод результата в консоль
    console.log('Результат сложения:', результат);

3. Посмотрите на результат в консоли:

После ввода кода нажмите Enter, и вы увидите результат сложения двух чисел в консоли браузера.
