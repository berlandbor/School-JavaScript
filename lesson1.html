# Зачем изучать программирование?

Изучение программирования полезно по нескольким причинам:

  • Автоматизация задач для экономии времени.
  • Развитие логического мышления и аналитических навыков.
  • Больше карьерных возможностей в сфере технологий.
  • Творческий потенциал в создании цифровых продуктов.
  • Понимание современных технологий и их использование.

# Что такое программирование?

Программирование — это процесс создания инструкций, которые компьютер может выполнить. Это позволяет автоматизировать задачи и создавать разнообразные программы для решения различных задач.

Знакомство с JavaScript

JavaScript - это язык программирования, который применяется для создания интерактивных веб-страниц.

1. Переменные и типы данных:

        
let имяПеременной = 'значение'; // объявление переменной
const постояннаяПеременная = 42; // константа

let число = 3.14; // число
let строка = "Привет, мир!"; // строка
let правдаИлиЛожь = true; // булево значение

2. Операторы:

        
let сумма = 5 + 3; // сложение
let разность = 10 - 4; // вычитание
let произведение = 2 * 6; // умножение
let частное = 8 / 2; // деление

3. Условные операторы:

        
let число = 10;

if (число > 5) {
    console.log('Число больше 5');
} else {
    console.log('Число меньше или равно 5');
}

4. Циклы:

        
for (let i = 0; i < 5; i++) {
    console.log('Шаг цикла:', i);
}

5. Функции:

        
function приветствие(имя) {
    console.log('Привет, ' + имя + '!');
}

приветствие('Мир');

6. Массивы:

        
let мойМассив = [1, 2, 3, 'строка', true];

console.log(мойМассив[0]); // вывод элемента массива

7. Объекты:

        
let мойОбъект = {
    ключ: 'значение',
    число: 42,
    isTrue: true
};

console.log(мойОбъект.ключ); // доступ к значению по ключу

8. События:

        
// HTML: 

document.getElementById('мояКнопка').addEventListener('click', function() {
    console.log('Кнопка была нажата!');
});

Этот урок представляет лишь введение в мир JavaScript и программирования. Дальнейшее изучение позволит вам создавать более сложные веб-приложения, взаимодействовать с пользователями и использовать современные технологии для решения различных задач.
